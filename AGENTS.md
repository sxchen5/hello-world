# AGENTS.md

## Cursor Cloud specific instructions

This repository includes a single-file personal dashboard:

- **`dashboard.html`** — static page (Tailwind CSS v3 CDN, Font Awesome). Open it in a browser (double-click or `open dashboard.html` / serve with any static file server). Data is stored in `localStorage`; no backend.

There is otherwise:

- **No application services** to run for the hello-world readme alone
- **No dependencies** to install for the dashboard (CDN assets load at runtime)
- **No build system**, linter, or test framework configured
- **No Docker or docker-compose** configuration

If more application code is added in the future, update this section with relevant build/run/test instructions.
